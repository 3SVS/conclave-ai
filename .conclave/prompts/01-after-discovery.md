# Claude Code Prompt — Stage 0 이후 계획 확정

Stage 0에서 작성한 `.conclave/docs/repo-findings.md`를 읽고, 다음을 수행하라.

1. Stage 1 구현 범위를 아주 작게 재정의한다.
2. 수정할 파일 목록을 제안한다.
3. 새로 만들 타입/상수/컴포넌트가 있다면 이름을 제안한다.
4. 예상 테스트를 제안한다.
5. 아직 구현하지 말고 사용자 확인을 기다린다.

출력 형식:

```text
Stage 1 구현 제안

목표
- ...

수정할 파일
- ...

새로 만들 파일
- ...

테스트
- ...

위험
- ...

확인 필요
- ...
```

### Short description (≤160자)
```
Drop a PRD and every PR auto-reviews against your spec — catching scope drift, missing requirements, and unauthorized changes that diff-only review can't see.
```

### Long description (≤500자)
```
Conclave AI is a GitHub App that reviews each pull request against your spec, not just the diff. Drop a .conclave/prd.md once; from then on every PR is automatically checked for scope drift, missing requirements, and unauthorized changes — the code that runs fine but isn't what you asked for. Three frontier models deliberate in parallel, then one verdict lands with an autofix worker that pushes the fix. Learns from your merge/reject decisions. Free on public repos, self-hostable under FSL. Not a Cursor/Copilot replacement — it's a merge gate at the PR layer.
```

## 실행 팁
- 폼마다 "AlternativeTo of?"를 물으면: CodeRabbit, Greptile, Qodo, Sourcery로 적기 → 경쟁사 검색 트래픽 일부 흡수.
- 대부분 승인까지 수일~2주 걸림. 내일 PH 런치 전에 1~3번이라도 제출해두면 런치 주에 노출 겹침.
- 로고/스크린샷은 marketing/assets/에 이미 있음. 그대로 업로드.

> 원하면 제가 Chrome으로 1~5번 폼 작성/제출까지 대신 진행 가능. (계정 생성·이메일 인증이 끼는 곳은 Bae가 직접 해야 하는 단계가 있을 수 있음 — 그 지점에서 멈추고 알려드림.)
